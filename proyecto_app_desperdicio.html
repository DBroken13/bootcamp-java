🌱 Proyecto Bootcamp – App Web para Control del Desperdicio Alimentario

🎯 Objetivo

Crear una app web sencilla, económica y escalable para que cocinas profesionales (restaurantes, hoteles, caterings, colegios, etc.) puedan registrar, analizar y reducir el desperdicio de alimentos de forma fácil y sin necesidad de sensores ni hardware.

🧑‍💻 Público objetivo

  • 🏨 Pequeñas y medianas empresas del sector HORECA
  • 🍽️ Cocinas con procesos manuales (Excel, hojas impresas)
  • 💸 Negocios que no pueden permitirse hardware como sensores o IA de visión

🚀 ¿Por qué esta app es necesaria?

  • 🔹 Muchos lugares tiran comida sin dejar constancia real
  • 🔹 Algunos usan papel o Excel → los datos se pierden o no se analizan nunca
  • 🔹 Herramientas como KITRO u Orbisk son caras o requieren hardware

👉 Nuestra app:

  • Registra fácilmente qué, cuánto y por qué se desperdició
  • Guarda los datos automáticamente en una base de datos
  • Permite analizar tendencias, enviar alertas y generar reportes
  • Ayuda a reducir costes, cumplir normativas y tomar decisiones

🧾 ¿Cómo se usa?

  1. 👨‍🍳 El operario tira comida → abre el panel desde una tablet o PC
  2. 📝 Selecciona el tipo de alimento, peso (o cantidad estimada) y el motivo
  3. 💾 Se guarda en la base de datos con fecha, turno y cocina
  4. 📊 El dashboard muestra gráficos y tendencias
  5. 📩 Se generan alertas y reportes automáticos según los datos registrados

📢 Sistema de Alertas Inteligentes

👨‍🍳 Operario

  • ⚠️ Recordá pesar la bandeja de comida al final del turno
  • ❗ No se registró desperdicio hoy

👨‍🍳 Jefe de cocina

  • 📈 Tercer día con exceso de arroz
  • 🔄 Revisión: devoluciones frecuentes con carne

👨‍💼 Gerente

  • 💸 Superado el coste mensual estimado
  • 🥩 Carne es lo más tirado esta semana

🏢 Dirección / Sostenibilidad

  • 📉 No se han registrado datos en 5 días
  • 📊 Ratio de desperdicio fuera de lo permitido
